# v0.1.136 - 登录保持与评论回复弹窗

## 更新内容

- 登录 Cookie 改为稳定快照存储，避免 WebKit/系统 Cookie 在退出 App 后没有可靠恢复。
- App 进入 inactive/background 时会主动同步并落盘登录 Cookie，减少被系统杀进程导致登录态丢失的概率。
- 登录状态刷新增加离线兜底：短暂网络/API 异常时不再直接清掉上次确认的登录状态。
- 评论回复从列表内联展开改为 iOS 原生 sheet 弹窗，支持拖拽关闭、中等/大尺寸停靠和下拉刷新。
- 回复弹窗独立分页加载，主评论列表不再保存大量展开回复，降低评论页滚动时的状态和渲染负担。

## 验证建议

- 登录账号后完全退出 App，再重新打开，确认个人页仍显示已登录状态。
- 进入视频详情页评论区，点击“查看 N 条回复”，确认以系统弹窗打开。
- 在回复弹窗内下拉刷新、加载更多、拖拽关闭，确认主评论列表位置不乱跳。
- 断网或接口临时失败后重新打开 App，确认不会立刻误清登录态。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
