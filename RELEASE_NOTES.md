# v0.1.141 - 修复详情页安全区与右滑返回

## 更新内容

- 修复视频详情页播放器顶部安全区：视频窗口上边缘不再越过状态栏底边。
- 修复“查看评论”入口命中区域：整块液态玻璃按钮区域都可以点击打开评论弹窗。
- 恢复全局右滑返回上一页手势，避免隐藏导航栏后系统返回手势失效。
- 首页、搜索页、我的页三个导航栈统一接入右滑返回恢复逻辑。
- 保留上一版视频详情页顶部固定播放器、液态玻璃卡片和评论弹窗不遮挡播放器的改造。

## 验证建议

- 进入视频详情页，确认视频窗口顶部从状态栏下方开始，不压住时间、电量等系统状态栏内容。
- 点击“查看评论”按钮的文字、图标、空白区域和边缘区域，确认都能打开评论弹窗。
- 从首页、搜索页、我的页进入任意二级页面，确认屏幕左边缘右滑可以返回上一页。
- 打开评论弹窗，确认弹窗顶部仍然不会覆盖视频窗口区域。

## 打包说明

- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
