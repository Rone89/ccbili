# v0.1.138 - 评论弹窗入口与详情信息精简

## 更新内容

- 移除视频详情卡片中“已自动切换清晰度”和“上次看到进度”的提示胶囊，减少信息噪音。
- 取消“简介/评论”分段切换关系，详情页默认直接展示简介内容。
- 新增独立“查看评论”按钮，点击后使用 iOS 原生 sheet 弹窗展示评论。
- 评论弹窗保留排序、下拉刷新、分页加载和查看楼中楼回复能力。
- 楼中楼回复弹窗挂在评论弹窗内部，避免多个 sheet 在详情页根层级互相抢占展示。

## 验证建议

- 打开视频详情页，确认圈选的两个提示胶囊不再显示。
- 确认详情页没有“简介/评论”分段切换，简介内容直接展示。
- 点击“查看评论”，确认评论以系统弹窗打开，并可关闭返回详情页。
- 在评论弹窗里切换排序、下拉刷新、加载更多、查看回复，确认交互正常。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
