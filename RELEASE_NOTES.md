# v0.1.142 - 修正详情组件起始位置

## 更新内容

- 修复视频详情页首个信息卡片初始位置过高的问题，详情组件上边缘不再超过播放器下边缘。
- 信息区、评论入口和简介内容从播放器下方开始排列，保留播放器固定在顶部的结构。
- 保留状态栏安全区、整块评论按钮可点击、全局右滑返回等上一版修复。

## 验证建议

- 进入视频详情页，确认视频信息卡片顶部低于视频播放器底部，不再压住视频画面。
- 上下滚动详情页，确认播放器仍固定在顶部，其他组件从播放器下方开始滚动。
- 点击“查看评论”整块按钮区域，确认评论弹窗正常打开且不覆盖播放器。
- 从详情页左边缘右滑，确认可以返回上一页。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
