# 本版本说明

## 新增功能

- 增强视频播放进度记忆，播放过程中会按进度百分比自动保存。
- 再次打开视频详情页时，会从上次观看位置继续播放。
- 历史页会显示“继续观看到 xx%”提示，并在封面底部显示进度条。

## 优化规则

- 播放进度低于 3% 时不保存，避免误触产生无意义记录。
- 播放接近完成时自动清除进度，避免已看完视频仍提示继续观看。
- 离开详情页时会强制保存当前进度，减少最后一次进度丢失。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
