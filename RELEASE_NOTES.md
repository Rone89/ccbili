# v0.1.148 - 详情页主容器改为评论弹窗同款玻璃

## 更新内容

- 视频详情页三个主容器改为和评论弹窗更接近的系统材质玻璃：先使用 `.regularMaterial` 做底，再叠加 iOS 26 原生 `.regular` Liquid Glass 高光。
- 评论入口保留交互态玻璃效果，点击手感不变，视觉层级和信息卡、简介推荐保持一致。
- 保留上一版评论弹窗、回复跳转修复、用户主页刷新移除、滑动性能和竖屏播放器控件修复。

## 验证建议

- 打开视频详情页，确认信息卡、评论入口、简介推荐三张卡的玻璃质感接近评论弹窗背景。
- 切换亮色/暗色内容视频，确认卡片透明度、文字可读性和圆角一致。
- 点击“查看评论”，确认评论弹窗、回复弹窗和竖屏播放器控件仍正常。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
