# v0.1.145 - 评论弹窗样式与播放器控件修复

## 更新内容

- 评论弹窗改为和“查看回复”一致的原生 NavigationStack + List 弹窗样式。
- 去掉点赞、收藏、投币、分享操作区外层底部容器，只保留按钮本身。
- 评论和回复的用户名使用白色，评论/回复正文使用灰色，预览回复也按用户名/正文分色显示。
- 去掉相关推荐单项中封面、标题下方的圆角矩形容器，推荐列表更轻量。
- 修复竖屏播放器系统控件丢失：竖屏内嵌播放时手势层不再拦截 AVPlayerViewController 原生控制条触摸，横屏全屏时仍保留手势层。

## 验证建议

- 点击“查看评论”，确认评论弹窗外观与“查看回复”的原生列表弹窗一致。
- 检查详情卡片内点赞/收藏/投币/分享区域，确认底部圆角容器已移除。
- 检查评论、预览回复、回复列表，确认用户名为白色、内容为灰色。
- 检查相关推荐，确认每个推荐项不再有单独的圆角底板。
- 竖屏播放视频时点击播放器，确认系统播放控件可以正常出现并可操作。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
