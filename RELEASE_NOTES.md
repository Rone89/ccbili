## 本版本说明

### 新增功能

- 播放器统一改为自定义覆盖层：关闭 AVPlayerViewController 自带控件，DASH/低清与高清链路使用同一套播放 UI。
- iOS 26 及以上使用 Liquid Glass API 呈现播放按钮、快进快退、全屏和进度条；旧系统自动回退到 `ultraThinMaterial`。
- 分享按钮接入系统分享面板，分享当前 B 站视频链接。

### 修复问题

- 播放量、上传日期和清晰度切换按钮统一为 34pt 高度，信息行视觉高度一致。
- 收藏按钮成功后同步更新收藏状态和收藏计数。
- 投币、点赞继续接入 B 站接口，分享计数在触发分享时同步增加。
- 保持 Release 说明为 UTF-8 无 BOM，并在发布前再次规范编码，避免中文乱码。
