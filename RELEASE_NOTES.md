# v0.1.140 - 视频详情页顶部播放器与液态玻璃界面

## 更新内容

- 移除视频详情页顶部导航返回按钮和“视频详情”标题，让视频窗口直接固定在页面顶部。
- 将视频窗口作为详情页底层固定区域，标题、UP 主信息、操作区、评论入口和简介内容覆盖在播放器上层滚动展示。
- 视频信息卡、清晰度入口、操作栏、评论入口、简介卡片、评论弹窗内部组件统一接入 iOS 26 Liquid Glass；旧系统保留 `.ultraThinMaterial` 兜底。
- 评论弹窗高度按播放器底部动态限制，弹窗顶部不会越过视频窗口底部，避免遮挡播放画面。
- 删除旧的独立标题卡片/作者卡片冗余结构，保留当前视频信息卡里的标题与作者内容。

## 验证建议

- 竖屏进入视频详情页，确认顶部没有返回按钮和“视频详情”标题，播放器贴近页面顶部。
- 上下滚动详情内容，确认信息卡、评论入口和简介位于播放器上层，播放器保持在底层顶部。
- 点击“查看评论”，确认评论弹窗从底部弹出且顶部不覆盖视频窗口区域。
- 在 iOS 26 上确认各卡片、按钮、评论弹窗呈现 Liquid Glass 质感；低版本 iOS 确认毛玻璃兜底正常。
- 播放视频时确认控制条、详情卡片和评论弹窗互不异常遮挡。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。
