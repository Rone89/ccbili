# v0.1.126 - 修复横竖屏切换仍会暂停

## 更新内容

- 修复竖屏转横屏时仍被暂停的问题。
- 修复横屏退回竖屏时仍被暂停的问题。
- 横竖屏切换期间不再把播放器从竖屏容器上强行摘掉，避免 AVKit 触发短暂停顿。
- SwiftUI 刷新期间不再把弹幕/手势层从全屏容器抢回竖屏容器，减少切换抖动。
- 外层播放器不再因为系统短暂进入 inactive 状态而暂停；只有真正进入后台时才暂停。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
- 未签名 IPA 需要自行重签名后安装到真机。

## 验证建议

- 竖屏播放视频，直接旋转到横屏，确认播放不中断、不变成暂停按钮。
- 横屏播放时旋回竖屏，确认仍在继续播放。
- 连续横竖屏切换多次，确认进度条持续前进。
- 测试 AVKit 原生播放器和普通播放器路径，确认两边都不再因旋转暂停。
