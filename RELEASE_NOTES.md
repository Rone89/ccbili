# v0.1.121 - 滑动性能优化

## 更新内容

- 优化首页推荐流的滚动性能，移除滚动过程中的高频状态刷新和批量动画。
- 优化远程图片加载，使用共享缓存、后台解码和缩略图尺寸限制，降低滑动时的主线程压力。
- 降低视频卡片的阴影、占位加载动画和分页触发开销，让列表滚动更稳定。
- 优化搜索结果列表渲染，视频和用户结果都减少滚动中的重复任务。
- 搜索页背景改为 iOS 系统分组灰色，避免纯白底色与系统页面不一致。

## 打包说明

- GitHub Actions 会构建 Release 配置的未签名 IPA。
- 生成的 `ccbili-unsigned.ipa` 会作为 Release 附件上传。
